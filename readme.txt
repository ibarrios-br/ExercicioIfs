Para esse exercício, utilizar somente condições IF, IF ELSE ou ELSE para resolução.

1 - Criar um console application que leia o primeiro nome de quem está usando.
2 - Ler o nome que foi inserido e guardar em uma variável.
3 - Limpar a tela do console e exibir o nome que foi salvo na variável, junto com o texto: "Olá, [Nome]. Vamos começar? S/N". 
4 - Ler a resposta do usuário, se for S, segue para o próximo passo, se for N, limpar o console e exibir o texto "[Nome], esperamos que volte em breve. Até logo!".
5 - Informar o usuário que será pedido uma sequência de 3 nomes e 3 idades de pessoas.
6 - Ler o primeiro nome e a primeira idade > Ler o segundo nome e a segunda idade > Ler o terceiro nome e a terceira idade.
6.1 - Nenhuma idade por ser igual a anterior, caso seja, informar o usuário que a idade precisa ser diferente das já informadas.
7 - Agora você precisa exibir os 3 nomes e idades por ordem crescente, do mais novo para o mais velho.
7.1 - No final da ordenação, informar o usuário para caso ele queira ordenar por ordem decrescente, aperta o botão ↓ ou ↑ para ordem crescente, ainda podendo ordenar aleatoriamente, apertando o botao →.
